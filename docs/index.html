FAQ

Q: How long will this work?

A: I don't know. Definitely not forever, but use it while you can!

Q: Will this work with Social Fixer?

A: Yes! All the features of Social Fixer should work with the Old Layout

Q: How does it work?

A: It forces all requests to www.facebook.com to go to upload.facebook.com instead. This url still serves the Old Layout.
