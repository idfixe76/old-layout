Old Layout Returns!

Changes in Version 6.0 (2020-10-23):

  • Old Layout now uses an entirely new approach. You will see a panel appear in the lower right corner of your Facebook page with a button to switch back.
  • This button triggers Facebook's internal "switch back" mechanism, so there are no quirks, no hacks, and hopefully no bugs
  • The switch will "stick" for 48 hours. Then the message will return and you can switch back again!
Please report any issues or suggestions to the Old Layout Page. I'm continuing to work on bugs and quirks in Facebook's old layout!
