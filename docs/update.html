Old Layout for Facebook has been "fixed"!

Changes in Version 5.1 (2020-10-20):

Facebook killed Old Layout for a while, but I found a new method to keep you in the Old Layout!

The new fix reroutes requests to www.facebook.com through upload.facebook.com, which still serves the old layout.

Hopefully this fix will stick around a while, but no guarantees.

Note: Messenger will still use the new layout, due to bugs when used on upload.facebook.com. This is unavoidable.

If this fix stops working as well, I will keep finding fixes as long as it's possible!!

Changes in Version 5.2 (2020-10-22):

Patched so Messages send correctly from facebook.com/messages/

Please report any issues or suggestions to the Old Layout Page. I'm continuing to work on bugs and quirks in Facebook's old layout!
